Fantasy Fishing

Fantasy Fishing is a Full Stack fishing game utilizing the external FishAPI as well as a backend SQL database which stores data. Users can create an account, cast their line out and pull one of 906 fish or nothing at all from the user chosen depth. There are trophies to be earned, buffs that enhance the user's fishing experience that can be purchased using in game currency, and a leaderboard to climb to the top of. There is also an in app music player, containing original music.

This project is still being updated so check back often.

Library Terminal

Myself and three other devs were tasked with creating a library terminal app where the user could check books in and out and track each books availability status. Using I/O functions inside of C#, we created a user system that allows a user to create their own user profile. Inside of said user profile checked out books would be stored in their name indvidually allowing specific users to check in their specific books. There was also permissions only granted to specific users. All of this was done in C# without the use of any database/SQL.

Check it out on GitHub

Help Desk Ticket System

A full stack online ticket system where a user can create a profile, submit an IT ticket to the system detailing an issue that they are facing. That ticket can then be closed out by an admin with a resolution message or saved in favorites for quick reference on recurring issues. Tickets are sorted by the end users ID.
