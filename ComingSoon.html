ESL Classroom App

A classroom interface I am building for an English teacher in South Korea.

Once completed, the plan is for the website to allow the teacher to create a classroom, enroll students in indvidual classrooms, then plan an esl game for it. The instructor will select one of several ESL games from those I've created, the content will be editable. Students will be able to play the game and their performance will be trackable for the teacher.

Resume Explorer

Explore my body of work in a unique way

Resume explorer as a project that will allow the user to explore my body of work and history in a fun rpg based explorer.

You are traveler who explores various aspects of my resume using buttons and text Move west to view my projects. From there you can select a specific project by moving in a dirtection. And pick again to interact with it's details. Go west to open the Github repo for it, north to watch a demo in browser, etc.
